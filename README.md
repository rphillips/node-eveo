node-eveo
=========

A simply Eve Online client for NodeJS.

Contributors
============

Feel free to submit pull requests on github.

Configure
=========

```shell
# Edit ~/.node-eveo
{
    "ID": "Your Eve ID",
    "Key": "Your Eve API Key"
}
```

```shell
# chmod 600 ~/.node-eveo
```

Tests
=====

To run the tests make sure the .node-eveo file is setup.

```shell
# make tests
```
